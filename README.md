A lightweight, single-file web app for tracking shared expenses between two people. No app store, no subscription — just a live URL you both open on your phone or laptop.
Built with vanilla HTML/CSS/JS and Supabase as the backend database.

✨ Features
1. Log expenses — record who paid, how much, what category, and how it should be split
2. Smart splitting — supports 50/50 shared expenses, or expenses that belong to just one person
3. Live balance — instantly see who owes whom and how much, updated every time an entry is added
4. Settle Up — one-tap settlement that records the payback and resets the balance to zero
5. Category breakdown — monthly spending breakdown by category (Dine Out, Grab, Groceries, etc.)
6. Expense history — filterable list by person, sorted by most recent
7. Shared in real time — both users see the same data; any entry either person adds is instantly visible to the other


🛠️ Tech Stack
1. Frontend: Vanilla HTML/CSS/JS
2. Database: Supabase (PostgreSQL)
3. Hosting: Github Pages
